
You can change the default values of other parameters to simulate different conditions. Refer to the config [train.py](train.py).


This is the official implementation for the paper "FL-OA: A Byzantine-Robust Federated Learning Framework for Intelligent Devices with Outsourced Auditing".

You can find the paper once it is published.


## Usage

You can easily run the code by executing [train.py](train.py).

### Environment

Our code does not rely on special libraries or tools, so it can be easily integrated with most environment settings. 

If you want to use the same settings as us, we provide the conda environment we used in `env.yaml` for your convenience.

### Dataset

The paper used SVHN CIFAR10   CIFAR100 CINIC datasets are available on `torchvision` and will be downloaded automatically.
